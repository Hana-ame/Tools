# Tools Collection

This repository contains a variety of utility tools and scripts.

## Quick Guide
Since the directory structure is flat, you can find tools by their name or purpose. 
If you are looking for a specific functionality, feel free to ask opencode.

### Key Tools
- **Image Tools**: Look for files starting with `!` or containing `webp`, `avif`, `jpg`, `png`.
- **Web Tools**: Look for `download_`, `uploader`, `post_tool`.
- **System Tools**: Look for `.sh`, `.ps1` scripts.
- **Development**: `agent_runner.py`, `command_listener.py`, etc.

For any questions about how to use a specific script, just ask!
